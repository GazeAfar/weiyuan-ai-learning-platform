# 微远AI学习平台 V1

这是一个面向 **中国江苏省南京市初三** 的轻量网站版本，当前先上线 **物理**，并已预留 **数学** 扩展位。物理默认教材版本为 **苏教版（江苏凤凰科技出版社）**，支持：

- 按学科、地区、教材版本、知识点、难度、题量调用 **真实 AI 出题**
- 在线测试与即时评分
- 显示正确答案与解题思路
- 错题自动加入错题集
- 从错题集中勾选题目，继续生成 **类似题** 强化练习
- 页面按学科与知识模块拆分，风格偏正式、简洁

## 目录结构

- `server.js`：Node 服务 + AI 接口代理 + 静态资源服务
- `public/index.html`：页面结构
- `public/styles.css`：页面样式
- `public/app.js`：前端交互逻辑
- `.env.example`：环境变量示例

## 启动方式

### 1）配置环境变量
可以先参考 `.env.example`：

```bash
export OPENAI_API_KEY='你的key'
export OPENAI_BASE_URL='https://api.openai.com/v1'
export OPENAI_MODEL='gpt-4o-mini'
export PORT=8787
```

如果你使用的是兼容 OpenAI 协议的服务，也可以替换 `OPENAI_BASE_URL` 和 `OPENAI_MODEL`。

### 2）启动

```bash
npm start
```

### 3）访问

打开：

```text
http://127.0.0.1:8787
```
