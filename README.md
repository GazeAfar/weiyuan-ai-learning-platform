# 微远AI学习平台 V1

这是一个面向 **中国江苏省南京市初三** 的轻量 Next.js 网站版本，当前已上线 **物理** 和 **化学**，并已预留 **数学** 扩展位。物理默认教材版本为 **苏教版（江苏凤凰科技出版社）**，化学默认教材版本为 **人教版**。

支持：

- 按学科、地区、教材版本、知识点、难度、题量调用 **真实 AI 出题**
- 化学按南京初中化学风格生成 **推断题**
- 在线测试与即时评分
- 显示正确答案与解题思路
- 错题自动加入错题集
- 从错题集中勾选题目，继续生成 **类似题** 强化练习
- 页面按学科与知识模块拆分，风格偏正式、简洁

## 目录结构

- `app/`：Next.js 页面与 API 路由
- `components/study-platform.tsx`：前端练习、错题集、打印交互
- `lib/curriculum.ts`：地区、教材版本、学科与知识点配置
- `lib/ai.ts`：AI 出题与错题相似题提示词

## 启动方式

### 1）配置环境变量

```bash
export OPENAI_API_KEY='你的key'
export OPENAI_BASE_URL='https://api.openai.com/v1'
export OPENAI_MODEL='gpt-4o-mini'
```

如果你使用的是兼容 OpenAI 协议的服务，也可以替换 `OPENAI_BASE_URL` 和 `OPENAI_MODEL`。

### 2）启动

```bash
npm run dev
```

### 3）访问

打开：

```text
http://127.0.0.1:3000
```

## 当前说明

- 当前不含登录、学习记录、家长端
- 当前物理、化学知识点已接入；数学仅预留学科入口，后续可补题纲
- 化学目前只生成推断题
- 错题集存储在浏览器 `localStorage`
- 判分逻辑目前为轻量规则判分；后续可升级成“AI 判分 + 步骤分”
